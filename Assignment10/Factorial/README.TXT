Compiling and loading continuation-based evaluators (Factorial/README.TXT)
--------------------------------------------------------------------------

A. Compile and run a Java implementation of factorial in
   continuation-passing style:

   javac Factorial.java
   java Factorial 10

   javac Factorial2.java
   java Factorial2 10


B. Compile and run a C# implementation of factorial in
   continuation-passing style:

   dotnet build
   dotnet run 10
